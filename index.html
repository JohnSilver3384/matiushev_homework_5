Тестовий сайт про зовнішій CSS

Тепер стилі живуть в окремому файлі

Чому це краще?

Використання окремого файлу дозволяє:

  • Чистота: HTML-код стає легшим для читання.
  • Перевикористання: Один CSS-файл можна підключити до багатьох сторінок.
  • Кешування: Браузер завантажує файл один раз, що пришвидшує сайт.
